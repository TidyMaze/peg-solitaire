# Peg-solitaire

A peg solitaire solver in scala.

Run it, it will solve the english board and print all solutions.

```
...

SUCCESS 9747!
##...##
##...##
.......
...o...
.......
##...##
##...##
```

Current speed after micro-optim: 2 000 000 state/s (one state is a game node, can be leaf as well).
